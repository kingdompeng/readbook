## 15.6 复制
别名会降低程序的可读性，一个地方的修改可能会对另一个地方造成预料之外的影响
通常用复制的方法取代为对象起别名
浅复制：仅复制对象以及其包含的引用，但未复制嵌套的对象
```python
import copy
box2 = copy.copy(box)
```
深复制：不仅复制对象，还可以复制这个对象所引用的对象，甚至可以复制这个对象所引用的对象的所引用的对象
```python
box3 = copy.deepcopy(box)
```
